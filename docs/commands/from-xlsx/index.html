from-xlsx

Parses MS Excel binary data into a table. open calls from xlsx automatically when the file extension is xlsx. Use this command when open is unable to guess the file type from the extension.

Examples

> open abc.xlsx
─────────────────
 Sheet1
─────────────────
 [table 26 rows]
─────────────────
> open abc.xlsx --raw
