select

This command displays only the column names passed on to it.

Examples

> ls
━━━┯━━━━━━━━━━━━━━━━━━━━━━━━━━━━┯━━━━━━┯━━━━━━━━━━┯━━━━━━━━┯━━━━━━━━━━━━━┯━━━━━━━━━━━━━┯━━━━━━━━━━━━━

───┼────────────────────────────┼──────┼──────────┼────────┼─────────────┼─────────────┼─────────────

 1 │ coww.txt                   │ File │          │  24 B  │ a month ago │ a month ago │ a month ago
